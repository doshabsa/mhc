What is Waves?

"Waves is a community-based stack of decentralized open-source technologies to build scalable, user-friendly apps"

Waves was create to to provide an open source platform for developers to create apps that utilize blockchain. They also designed it so that it minimizes the carbon footprint that can be an issue with other blockchain technologies.

Because it is open source, this project is availiable for anyone to take a part in developing. This is a great way for those who want to get more involved to do so, and give an oppourtunity to those who want to learn more about the technology.
